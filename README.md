# my-portfolio

Portfolio of data science and AI projects focused on supply chain management, demand forecasting, optimization, and educational tools.

## Projects

- [TROEL OPS Kit](./TROEL%20OPS%20Kit/README.md): Python CLI for ingest, validation, KPI calculation, alerts, and reporting on synthetic supply-chain data.

## Repository layout

- Each project lives in its own folder.
- GitHub Actions workflows live at the repository root in `.github/workflows/` because this is a portfolio repo, not a single-project repo.
