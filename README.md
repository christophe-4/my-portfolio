## 🔹 Projet 1 – Classification avec scikit-learn

- 📂 Dossier : [`projects/scikit-learn-basics`](./projects/scikit-learn-basics)  
- 🧩 **Idée générale** :  
  Mettre en place un pipeline complet de machine learning classique (prétraitement → entraînement → évaluation) sur un jeu de données tabulaire (par ex. prédiction d’un churn client, d’une catégorie de produit, etc.).

**Points clés :**
- Nettoyage & préparation des données
- Séparation train/test, validation croisée
- Comparaison de plusieurs algorithmes (LogisticRegression, RandomForest, XGBoost, etc. selon ce que tu utilises)
- Interprétation simple des résultats (accuracy, f1-score, matrice de confusion)

## 🔹 Projet 2 – AutoGluon pour la prévision de la demande

- 📂 Dossier : [`projects/autogluon-demand-forecast`](./projects/autogluon-demand-forecast)  
- 🧩 **Idée générale** :  
  Utiliser **AutoGluon** pour automatiser la modélisation (AutoML) sur un problème de prévision ou de régression (par exemple : prévision des ventes sur 30 jours).

**Points clés :**
- Description du jeu de données (colonnes, cible, horizon de prévision)
- Configuration d’AutoGluon (tabular ou time series)
- Comparaison avec une baseline simple
- Discussion : ce que l’AutoML apporte en termes de rapidité / performance
